HTML

  • The head element contains unseen meta-data concerning the page (both for the browser and other developers)
    • The title element defines the title for a webpage. This element is required and is extremely important for Search Engine Optimization (SEO).
    • The style element defines the CSS styles associated with a website.
    • The base element defines the base URL for a webpage.
    • The link element connects an external resource to the HTML document.
    • The meta element defines metadata such as the character set, description, keywords, author, and viewport.
    • The script element embeds client-side scripts in the HTML document.
  • The body element contains the main content of the page that is seen
    • The header element appears at the top of the page and usually contains a navigation menu and/or introductory content
    • The main element where the main content is located
      • The image element is used to add images to a website
      • The paragraph element allows us to wrap up a text
      • The form element allows one to capture input and data from the web browsers
      • The listing element allows one to create lists, in either the ordered or unordered manner (ordered lists generally follow an important order while unordered lists do not)
      • The anchor element is used to either make links to another site, or different pages on the same site
    • The footer appears at the bottom of the page and usually contains the author, copyright, contact, sitemap, and navigation.

CSS

  • There are three main ways to style using CSS
    • Inline styling, which is to inject CSS code right into the element itself using the style attribute
    • Internal style sheet, which is to use a selector tied to a declaration (or multiple declarations becoming a declaraton block)
    • External style sheet, which is to attach a seperate CSS code file through a link element
  • CSS uses properties in order to add style, which are then paired to a value of said property
    • Background-color property changes the background color of an element
    • Border property adds a border to an element
    • Box-shadow property adds a box to an element
    • Color property adds color to a text
    • Display property specifies the display behaviour
    • Font-size property adusts the size of the font
    • Height property adjusts the height of an element
    • Margin property adjusts the margin of an element
    • Padding property adjusts the padding of an element
    • Text-align property adjusts the alignment of text
    • Width property adjusts the width of an element

Git

  • Git status: command that lets us know which branch we are currently working in
  • Git checkout -b "name of branch": command that creates a new branch and moves to it
  • Git add -A: command that adds current modifications in current working branch into the staging area (the -A flag says that we want to add all changes)
  • Git pull origin main: command to pull the main branchs data into the local environment (to sync the local branch with the repo branch)
  • Git push origin "name of local branch": command to push the changed data of the local branch onto the repo branch
